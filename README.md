# HackerEarth
#### Setup Steps
##### 1. Install all the dependencies with  
```
pip install -r requirements.txt
```

##### 2. Migrate
```
python manage.py migrate
```
##### 3. Create Super User

```
python manage.py createsuperuser
```

##### 4. Runserver
```
python manage.py runserver
```

Go to http://127.0.0.1:8000/admin
